    `List` (GET):
    ```go
    // List godoc
    // @Summary      List transaction templates
    // @Description  Returns the authenticated user's saved transaction templates, oldest first (max 3)
    // @Tags         transaction-templates
    // @Produce      json
    // @Security     CookieAuth
    // @Security     BearerAuth
    // @Success      200  {array}   domain.TransactionTemplate
    // @Failure      401  {object}  middleware.ErrorResponse
    // @Router       /api/transaction-templates [get]
    func (h *TransactionTemplateHandler) List(c echo.Context) error {
        userID := appcontext.GetUserIDFromContext(c.Request().Context())
        templates, err := h.templateService.List(c.Request().Context(), userID)
        if err != nil {
            return HandleServiceError(err)
        }
        return c.JSON(http.StatusOK, templates)
    }
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→handler | untrusted JSON body + path params cross here; a `user_id` in the body must be ignored |
| handler→service | handler supplies userID from the validated JWT (appcontext), never the body |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-27-01 | Spoofing / Elevation (IDOR) | all 4 handlers | mitigate | `userID` read from `appcontext.GetUserIDFromContext` and passed as the service argument; body `user_id` (if any) ignored. Handler test asserts the service receives the CONTEXT userID even when the body carries a different one. |
| T-27-03 | Information Disclosure (403 vs 404) | Update/Delete | mitigate | Owner mismatch surfaces the service/repo `NotFound` unchanged → 404 via HandleServiceError; handler never re-maps to 403. Swagger documents 404 for /:id routes. |
| T-27-07 | Information Disclosure (lost error tags) | error path | mitigate | `HandleServiceError` (tag-preserving) used for all service errors — NOT `echo.NewHTTPError(...)` which strips Tags. Grep gate bounds echo.NewHTTPError to param/bind guards only. |
| T-27-08 | Denial of Service (bad path param) | Update/Delete | mitigate | `strconv.Atoi` + `id <= 0` guard returns 400 before touching the service. |
</threat_model>

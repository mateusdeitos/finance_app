**Exported types and interface:**

```tsx
export interface BulkProgressItem {
  id: number
  label: string
}

interface BulkProgressDrawerProps {
  items: BulkProgressItem[]
  action: (item: BulkProgressItem) => Promise<void>
  titles: {
    processing: string
    success: string
    error: string
  }
  successMessage: (count: number) => string
  onInvalidate: () => void
  onSuccess: () => void
}
```

**Component structure:** Follow BulkDeleteProgressDrawer exactly for:
- `useDrawerContext<void>()` for drawer context
- State: `useState<"processing" | "success" | "error">("processing")`, `useState(0)` for progress, `useState("")` for currentLabel, `useState<ErrorInfo | null>(null)` for error info
- ErrorInfo type: `{ description: string; reason: string; remaining: string[] }`
- `useEffect` with sequential for-loop calling `action(items[i])`, setting progress as `Math.round((i / items.length) * 100)`, setting currentLabel as `items[i].label`
- Error extraction: identical pattern -- `if (err instanceof Response)` try `err.json()` for `body.message`, fallback `Erro ${err.status}`, final fallback `"Erro desconhecido"`
- On error: `setErrorInfo({ description: items[i].label, reason, remaining: items.slice(i + 1).map(t => t.label) })`, `setState("error")`, return
- On complete: `setProgress(100)`, `setCurrentLabel("")`, `setState("success")`, call `onInvalidate()`, call `onSuccess()`
- ESLint disable comment on useEffect deps: `// eslint-disable-next-line react-hooks/exhaustive-deps`

**Drawer props:**
- `position="bottom"`
- `data-testid` dynamic: use `bulk_progress_${state}` pattern -- but since the drawer serves multiple action types, use generic testids per UI-SPEC: the calling code will wrap this. Actually per UI-SPEC, data-testid should be passed in. Add an optional `testIdPrefix?: string` prop, default `"bulk_progress"`. Then: `data-testid={\`${testIdPrefix}_${state}\`}`
- `withCloseButton={!isProcessing}`
- `onClose={isProcessing ? () => {} : close}`
- `closeOnEscape={!isProcessing}`, `closeOnClickOutside={!isProcessing}`
- Title: `state === "success" ? titles.success : state === "error" ? titles.error : titles.processing`
- Bottom drawer styles: `content: { borderRadius: "var(--mantine-radius-lg) var(--mantine-radius-lg) 0 0", height: "auto", maxHeight: "80dvh" }`

**Processing state UI:**
- `<Progress value={progress} animated={isProcessing} color={state === "error" ? "red" : "blue"} data-testid="bulk_progress_bar" />`
- `<Text size="sm" c="dimmed" ta="center" data-testid="bulk_current_label">{currentLabel}</Text>`

**Dynamic text mapping based on actionLabel:**

- Title: `"Excluir transacoes recorrentes"` (current value, keep as-is including existing accents)
- Body: `"Algumas transacoes selecionadas fazem parte de uma serie recorrente. Como deseja exclui-las?"` (current value)

  - all: `"Exclui todas as recorrencias da serie"` (current)
- Confirm button: `"Confirmar exclusao"` (current), `color="red"`

When `actionLabel === 'alterar'`:
- Title: `"Atualizar transacoes recorrentes"` (per Copywriting Contract in UI-SPEC)
- Body: `"Algumas transacoes selecionadas fazem parte de uma serie recorrente. Como deseja atualiza-las?"` (per UI-SPEC)
- Radio descriptions:
  - current: `"Altera apenas a transacao selecionada"`
  - current_and_future: `"Altera esta e todas as recorrencias futuras"`
  - all: `"Altera todas as recorrencias da serie"`
- Confirm button: `"Confirmar alteracao"` (per UI-SPEC), `color="blue"` (primary, not destructive)
- Confirm button data-testid: `"btn_propagation_confirm_update"` (per UI-SPEC)

**Implementation approach:** Create a `copy` object derived from `actionLabel`:

```tsx
const isDelete = actionLabel === 'excluir'

const copy = {
  title: isDelete ? 'Excluir transacoes recorrentes' : 'Atualizar transacoes recorrentes',
  body: isDelete
    ? 'Algumas transacoes selecionadas fazem parte de uma serie recorrente. Como deseja exclui-las?'
    : 'Algumas transacoes selecionadas fazem parte de uma serie recorrente. Como deseja atualiza-las?',
  options: [
    { value: 'current' as const, label: 'Somente esta', description: isDelete ? 'Exclui apenas a transacao selecionada' : 'Altera apenas a transacao selecionada' },
    { value: 'current_and_future' as const, label: 'Esta e as proximas', description: isDelete ? 'Exclui esta e todas as recorrencias futuras' : 'Altera esta e todas as recorrencias futuras' },
    { value: 'all' as const, label: 'Todas', description: isDelete ? 'Exclui todas as recorrencias da serie' : 'Altera todas as recorrencias da serie' },
  ],
  confirmLabel: isDelete ? 'Confirmar exclusao' : 'Confirmar alteracao',
  confirmColor: isDelete ? 'red' : undefined, // undefined uses Mantine primary (blue)
  confirmTestId: isDelete ? 'btn_propagation_confirm' : 'btn_propagation_confirm_update',
}
```

Then use `copy.title` in Drawer title, `copy.body` in the Text, map `copy.options` for radios, and `copy.confirmLabel`/`copy.confirmColor`/`copy.confirmTestId` for the button.
